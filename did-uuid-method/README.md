# did:uuid Method Specification

Source for the W3C DID Method Specification that names RUUID's UUID
documents as DID documents and the RUUID resolution pipeline as the
DID resolver.

**[Read the rendered spec](https://worldresolvable.github.io/ruuid-draft/did-uuid-method/)**
— ReSpec renders the `index.html` as a W3C-style Unofficial Draft.

Companion documents:

  * The RUUID Internet-Draft this method is built on, as
    [HTML with anchors](https://worldresolvable.github.io/ruuid-draft/draft-motters-ruuid-00.html)
    or
    [canonical text](https://worldresolvable.github.io/ruuid-draft/draft-motters-ruuid-00.txt).
    The method spec's `Read` operation refers to its resolution pipeline
    directly.
  * The [Python reference implementation](https://github.com/WorldResolvable/ruuid-tools);
    the resolver is at `ruuid/did.py`.

This document is intended to be a candidate for the [W3C DID Method
Registry](https://www.w3.org/TR/did-spec-registries/#did-methods).
